# SpringBoot整合Netty进行功能实现

## Netty简介
  Netty 是一款用于快速开发高性能的网络应用程序的 Java 框架。它封装了网络编程的复杂性，使网络编程和 Web 技术的最新进展能够被比以往更广泛的开发人员接触到。
Netty 不只是一个接口和类的集合；它还定义了一种架构模型以及一套丰富的设计模式。
